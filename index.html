Hello World?

2021-02-01

本文所有内容仅代表本人在本文发表那一时间点的主观看法，
想法如有雷同算我抄您，如有不同您说的对！

#include <stdio.h>
int main()
{
printf(“Hello, World!”);
return 0;
}

为什么是Hello World呢？

想到这个问题的原因

从我开始接触程序，无论是什么语言，第一个接触到的代码永远是Hello World！
所以我就在想，为什么是Hello World？
为什么不是日常中很常见的How are you？Hey！Wussup！等等？

起源

于是，我去搜索了一些相关资料，发现这个短语第一次出现在书中，应该是在1972年由Brian Kernighan编写的《Introduction to the Language B》中
后来在1978年由Brian Kernighan和Dennis Ritchie编写的《The C Programme Language》使用而被广泛流行

但是作为作者本人，在网上最广泛的说法是，他自己也记不清为什么选择『Hello, World!』，印象中是在一个卡通片中听到一个小鸡说了句『Hello, World!』于是就使用了它。

总结

其实任何一个伟大的事物诞生，都可能只是一个巧合，就像hello world，或许只是动画片里的一句话罢了
但是这个语句对于计算机语言的意义，不亚于罗布泊的那一朵蘑菇云或者1970年的那一首东方红

仔细想想，这句话好像我们刚出生对这个世界打的招呼，代表了最纯粹，最简单的向往和探索
代表了一个生命的开始和初心，从这一步起，我们才有了探索无穷奥秘的开始
